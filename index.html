Conhecimentos

HTML CSS JavaScript Bootstrap
HTML (HyperText Markup Language) é a linguagem fundamental para a criação de páginas da web. Ela funciona através de etiquetas que envolvem o conteúdo, permitindo a organização de elementos como títulos, parágrafos, imagens e links. O HTML age como a estrutura básica de uma página, definindo a hierarquia e o tipo de informações presentes.
CSS (Cascading Style Sheets) é responsável pelo estilo e aparência das páginas da web. Com o CSS, você pode personalizar a cor, o tamanho e a fonte do texto, além de controlar o posicionamento e o espaçamento dos elementos na página. Ele permite que os desenvolvedores criem páginas visualmente atraentes e coerentes.
JavaScript (JS) é uma linguagem de programação usada para tornar as páginas da web interativas e dinâmicas. Com o JS, é possível adicionar funcionalidades como animações, validação de formulários e respostas a eventos do usuário, como cliques e movimentos do mouse. Ele desempenha um papel crucial na criação de experiências de usuário envolventes na web.
Bootstrap é um framework de front-end que simplifica o processo de criação de páginas da web responsivas e estilizadas. Ele fornece um conjunto de componentes, estilos pré-definidos e layouts flexíveis que facilitam o desenvolvimento. O Bootstrap é uma escolha popular entre os desenvolvedores para economizar tempo e garantir que as páginas sejam consistentes e compatíveis com dispositivos diversos.
